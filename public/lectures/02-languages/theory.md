# Лекция 02 — Языки программирования и исполнение кода

## Темы

- Языки программирования
- Исходный код, машинный код, байткод
- Способы интерпретации кода
- Runtime
- Типизация в языках программирования

## Короткий конспект

- Исходный код преобразуется в машинный код напрямую (компиляция) или через промежуточный байткод.
- Интерпретатор исполняет команды шаг за шагом, виртуальная машина исполняет байткод.
- Runtime — среда выполнения, которая предоставляет API, управление памятью и т.д.
- Типизация описывает правила работы с данными: статическая/динамическая, сильная/слабая.

## Про JIT и бенчмарки

- Если пишете бенчмарк, сравнивайте работу без JIT или одинаково прогревайте все тест кейсы.
- Например, Node.js можно запустить без JIT с флагом `node --jitless`.
